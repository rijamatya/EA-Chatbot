# Electricity Authority Chatbot

A Streamlit-based chatbot that answers user queries about the Electricity Authority using structured data (JSON/PDF) and an AI fallback via Ollama.

## Features

- Interactive chatbot interface using Streamlit
- Supports structured data (JSON) and unstructured data (PDF)
- Quick responses using keyword matching
- AI-powered fallback using Ollama (phi3 model)
- Maintains chat history during a session

## Tech Stack

- Python
- Streamlit
- Ollama (phi3)
- JSON / PDF data handling
- (Optional) LangChain for advanced PDF search
